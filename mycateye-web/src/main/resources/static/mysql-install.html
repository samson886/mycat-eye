    本系统提供MySQL5.7.17（目前最稳当、性能最好的版本）自动化脚本安装的方法 : 下载安装包
1、在服务器上创建/data/tools/mysql目录；
2、使用unzip mysql-5.7.17-setup.zip命令，将安装包中的install.sh、my.cnf、mysql-5.7.17-linux-glibc2.5-x86_64.tar.gz解压到/data/tools/mysql目录；
3、执行./install.sh；
4、等待安装，完成后，临时密码存放在/tmp/mypass.txt中。
5、使用临时密码登录到MySQL服务器，并使用set password='new password'，修改成自己的新密码。
6、完成初始安装
7、如需定制MySQL配置文件，请点击：自动生成my.cnf文件
